# Input新增类型

* `email`
* `url`
* `tel`
* `number`
* `range`
* `search`
* `color`
* `datePickers`
  * `Date` —— 选取日、月、年
  * `Month` —— 选取月、年
  * `Week` —— 选取周、年
  * `Time` —— 选取时间（小时和分钟）
  * `DateTime` —— 选取时间、日、月、年（UTC时间）
  * `DateTime-Local` —— 选取时间、日、月、年（本地时间）

**注意**:

`DateTime` 和 `DateTime-local` 区别

* 兼容性
* 一个本地时间，一个UTC时间
